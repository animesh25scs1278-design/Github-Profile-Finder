# Github-Profile-Finder
A responsive GitHub Profile Finder built with HTML, CSS, and JavaScript using the GitHub REST API, featuring async/await, Fetch API, and dynamic DOM manipulation.
This project was built using **HTML, CSS, and JavaScript** to strengthen my understanding of DOM manipulation, asynchronous JavaScript, Fetch API, and REST APIs.

## Features:
- 🔍 Search any public GitHub username
- 👤 Display user profile picture

- 📝 Display bio
- 📍 Display location
- 🏢 Display company (if available)

- 👥 Show followers, following, and public repositories
- ⌨️ Press **Enter** to search

## How It Works??
1. User enters a GitHub username.
2. JavaScript sends a request to the GitHub REST API.
3. The API returns user information in JSON format.
4. The application extracts the required data.
5. The profile card is dynamically updated using DOM manipulation.
6. If the username does not exist, an error message is displayed.
